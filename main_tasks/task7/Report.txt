Вне зависимости от количиства элементов в листе время работы сним занимало в среднем 15-20 миллисекунд.
Но для вектора количество элементов имеет значение: при 200 элементах среднее время работы 50-15 мс, при 2000 элементах - 50-150 мс.
Это связано с тем, что листе, чтобы добавить элемент нужно только создать новую ноду и соеденить с концом, а для удаления просто удалить ноду. Естесственно время этих действий не зависит от кол-ва элементов в листе.
Для вектора нужно сдвинуть каждый элемент массива, чтобы освободить первую ячейку и сдвинуть обратно все элементы, чтобы удалить элемент из первой ячейки. Чем больше элементов сдвинуть, тем дольше работает программа.
shared_ptr - позволяют хранить несколько ссылок на объект.
unique_ptr - только для хранения одной ссылки. Удобнее, если такой объект приватный в каком-нибудь классе, значит ссылок на него делаться не будет.
